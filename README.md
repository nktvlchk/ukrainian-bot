# 🇺🇦 Український Словник — Telegram бот

Телеграм-бот для вивчення української мови: словник (укр↔рос) з AI-перекладом та підручник з 14 тем граматики.

## Можливості

- **📖 Словник** — надішліть будь-яке слово, бот перекладе (укр↔рос) та покаже граматику
- **📚 Підручник** — 14 тем: абетка, займенники, числівники, час, іменник, всі 7 відмінків, професії, країни, одяг, по батькові
- **📋 Відмінки** — зведена таблиця всіх 7 відмінків з закінченнями
- **⭐ Обране** — збереження слів для повторення
- **❓ Вікторина** — міні-тест на знання граматики
- **🤖 AI-переклад** — Gemini перекладає слова, яких немає в базі

## Встановлення

### 1. Створіть бота в Telegram

1. Відкрийте [@BotFather](https://t.me/BotFather) в Telegram
2. Надішліть `/newbot`
3. Введіть ім'я бота, наприклад: `Український Словник`
4. Введіть username бота, наприклад: `ukr_slovnyk_bot`
5. Скопіюйте токен

### 2. Отримайте Gemini API ключ (безкоштовно)

1. Перейдіть на https://aistudio.google.com/apikey
2. Створіть API ключ
3. Скопіюйте його

### 3. Налаштуйте проект

```bash
cd telegram_bot

# Створіть .env файл
cp .env.example .env

# Відкрийте .env та вставте ваші ключі:
# TELEGRAM_BOT_TOKEN=ваш_токен_бота
# GEMINI_API_KEY=ваш_ключ_gemini

# Встановіть залежності
pip install -r requirements.txt

# Запустіть бота
python bot.py
```

## Команди бота

| Команда | Опис |
|---------|------|
| `/start` | Привітання та інструкція |
| `/textbook` | Підручник — 14 тем граматики |
| `/cases` | Таблиця відмінків |
| `/favorites` | Збережені слова |
| `/quiz` | Вікторина |
| `/help` | Допомога |

## Структура файлів

```
telegram_bot/
├── bot.py              # Головний файл бота
├── data.py             # Вся граматика та лексика (14 тем)
├── requirements.txt    # Залежності Python
├── .env.example        # Шаблон конфігурації
├── .env                # Ваші ключі (створіть з .env.example)
├── favorites.json      # Збережені слова (створюється автоматично)
└── README.md           # Цей файл
```
